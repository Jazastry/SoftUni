What is SQL?
	- SQL or 'Structured Query Language' is a special-purpose programming language designed for managing data held in a relational database management system.
What is DML? 
	- A data manipulation language (DML) is a family of syntax elements similar to a computer programming language used for selecting, inserting, deleting and updating data in a database.
What is DDL? 
	- A data definition language or data description language (DDL) is a syntax similar to a computer programming language for defining data structures, especially database schemas
Most important SQL commands
	- Data Manipulation Language (DML) commands
		SELECT, INSERT, UPDATE, DELETE
	- Data Definition Language (DDL) commands
		CREATE, DROP, ALTER
	- Data Control Language (DCL) commands
		GRANT, REVOKE
